# Buhari Segun | Calculator App

## Overview

Hey! This is my calculator project for the CMP module Git assignment. I used Expo and React Native to put it together. It's a pretty standard calculator but I added some things to make it better — like showing the equation you're building at the top of the screen, and making sure decimal calculations don't give weird results.

## Colour Theme

I went with a rose/red colour scheme. Felt like a strong choice that stands out from the default.

## How to Run

You'll need Node.js on your machine. Then just run these two commands:

```bash
npm install
npx expo start
```

Scan the QR code with Expo Go (Android) or the Camera app (iPhone).
